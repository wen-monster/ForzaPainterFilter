针对目前适用于 Forza:Horizon 的基础图形拟合图像项目普遍缺乏注意力机制，在背景等非重点区域浪费过量图层的问题，开发此交互式图层选择删除工具。

项目基于C++ Qt，目前仅支持 ForzaDesigner6 格式的JSON文件（个人认为 forza-painter 拟合效果优于 ForzaDesigner6 ，但gpu带来的速度优势还是太大了）。

基本操作：

1、点 Open 选择生成的JSON文件，左侧显示拟合图像，右侧为图层筛选参数；
![](https://github.com/wen-monster/ForzaPainterFilter/blob/main/image/%E5%B1%8F%E5%B9%95%E6%88%AA%E5%9B%BE%202026-06-14%20011511.png?raw=true)
2、直接点击图像，会选中点击位置最顶层图层，筛选参数不生效；

3、按 x 进入多边形框选模式，左键添加拐点绘制多边形，双击左键结束绘制，右键删除绘制中的多边形，选中中心在多边形内的图层，

可调节参数，反选图层、限制选中图层生成顺序范围、限制选中图层大小范围，

再次按 x 退出多边形框选模式，绘制的多边形和选中的图层均会取消；
![](https://github.com/wen-monster/ForzaPainterFilter/blob/main/image/%E5%B1%8F%E5%B9%95%E6%88%AA%E5%9B%BE%202026-06-14%20011552.png?raw=true)

4、选中图层黄色高亮，按 delete 删除，支持 Ctrl+Z 撤销和 Ctrl+Shift+Z 重做；
![](https://github.com/wen-monster/ForzaPainterFilter/blob/main/image/%E5%B1%8F%E5%B9%95%E6%88%AA%E5%9B%BE%202026-06-14%20011749.png?raw=true)
5、点 Save 将处理结果保存回JSON文件，可被原文件生成程序正常打开。

To address the common lack of attention mechanisms in basic graphics matching photos projects for Forza:Horizon, which waste excessive layers on non-critical areas such as backgrounds, this interactive layer selection and deletion tool has been developed.
The project is based on C++ Qt and currently only supports JSON files generated by ForzaDesigner6 (personally, I believe forza-painter provides better fitting results than ForzaDesigner6, but the speed advantage brought by GPU is still too significant).

Basic operations:
1. Click Open to select the generated JSON file. The left side displays the matched image, and the right side shows the layer filtering parameters.
2. Clicking directly on the image selects the topmost layer; filtering parameters do not take effect.
3. Press X to enter polygon selection mode. Left-click to add vertices and draw a polygon. Double-left-click to finish drawing and select the layers whose centers lie inside the polygon. Right-click to clear currently drawing polygon.
You can adjust parameters to invert selection, to restrict the generation order range of selected layers, and to restrict the size range of selected layers.
Press X again to exit polygon selection mode; both the drawn polygon and the selected layers will be cleared.
4. Selected layers are highlighted in yellow. Press Delete to remove them. Supports Ctrl+Z to undo and Ctrl+Shift+Z to redo.
5. Click Save to write the processed result back to the JSON file, which can be opened normally by the original program that generated the file.
